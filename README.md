# How to play?

## Download

	git clone https://github.com/GuidoPaul/2048.git

## Start

	cd 2048
	python 2048.py

## Play
You only need to press the 'a'(Left), 'w'(Up), 'd'(Right) or 's'(Down), then press "Enter".

Yes, It really is that simple.
