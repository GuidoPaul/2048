![Play 2048](/images/2048.png)

# How to play?

## Download

	git clone git@github.com:GuidoPaul/2048.git

## Start

	cd 2048
	python 2048

## Play
You only need to press the 'a'(Left), 'w'(Up), 'd'(Right), 's'(Down), then press "Enter".

Yes, It really is that simple.
